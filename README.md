# CivCity: Rome Mods

Este é um projeto aberto e colaborativo criado de fã para fã, todos são bem-vindos para ajudar e contribuir.

## Sobre o Projeto

O objetivo do projeto é oferecer modificações para o jogo CivCity: Rome focado no público brasileiro. Desde legenda do jogo até possivelmente uma dublagem futura.

A tradução atualmente está sendo feita sem revisão, após publicada fica aberto para quem quiser contribuir revisando.
